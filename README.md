## Quick Start

```bash
npm install
npm run dry-run
```

Create `.env` from `.env.example`, add your Discord webhooks, then run:

```bash
npm start
```

For automatic free deployment, use GitHub Actions and set these repository secrets:

```txt
EPIC_DISCORD_WEBHOOK_URL
STEAM_DISCORD_WEBHOOK_URL
```
